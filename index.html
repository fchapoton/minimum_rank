Minimum Rank Sage Library

See the example Sage worksheet for an example of how to load and use this library in Sage. In particular, the following code in a Sage notebook cell will load this library without needing to explicitly download anything.

URL='http://github.com/jasongrout/minimum_rank/raw/minimum_rank_1_1_1/'
files=['Zq_c.pyx','Zq.py','zero_forcing_64.pyx','zero_forcing_wavefront.pyx','minrank.py']
for f in files:
    load(URL+f)
